What's the Difference?

HTML

Hypertext Markup Language

Create the structure

.Controls the layout of the content
.Provides structure for the web page design
.The fundamental building block of any web page


CSS

Cascading Style Sheet

Stylize the website

.Applies style to the web pages elements .Targets various screen sizes to make web pages responsive .Primarily handles the "look and feel" of a web page


JavaScript

Increase interactivity

.Adds interactivity to a web page .Handles complex functions and features .Programmatic code which enhances functionality
